# Contributing

Thank you for your interest in contributing to `tv-lrud-spatial` (community fork)!

## Reporting Issues
Please check for existing [issues or Pull Requests](https://github.com/hps1978/tv-lrud-spatial/issues) before opening a new one.

If you cannot find an existing issue, create a new one and provide as much detail as possible.

## Contributing Code
Open an issue to discuss significant changes before submitting a pull request.

Fork this repository, create a feature branch, and submit a pull request with your changes.

Ensure your code is thoroughly tested.

## Pull Request Process
1. Complete all relevant sections of the PR template.
2. PRs can be merged once reviewed and approved by a maintainer.
